# Text/CSV Viewer

## Overview

This project is a Text/CSV Viewer developed in C# Windows Forms.

## Features

- Read text files
- Read CSV files
- Partial Loading (Start–End)
- File Type Filtering
- Support for large MalwareBazaar CSV dataset

## How to Use

1. Browse a CSV file.
2. Enter Start and End.
3. Enter File Type (optional).
4. Click Load CSV.

## Dataset

MalwareBazaar dataset is used for educational purposes only.

The full dataset is **not** included in this repository.
